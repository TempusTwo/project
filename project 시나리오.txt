
▣ 회 원 가 입 

   메인페이지에서 회원가입버튼을 누른다

   ①누를 시 회원가입 페이지로 이동한다.

    ②e-mail 인증을 한다.
        ㉠ e-mail 인증 전에는 다른 기능을 누를 시 경고창이 발생한다.

        ㉡메일 주소 입력 후 인증버튼을 누르면 본인메일로 인증메일이 발송

        ㉢해당 이메일의 링크를 클릭 시 인증 확인과 함께 다시 기입페이지로 돌아온다.

        ㉣열린페이지에는 작성했던 email주소가 기입되어있으며 이메일 기입칸은 비활성화되어 수정이 불가능하다.

    ③아이디 중복검사를 한다.
        ㉠ 아이디 중복검사 버튼을 클릭힌다.

        ㉡ 중복이라면 경고창이 뜨며 중복이 아니라면 사용가능하다는 알림이 뜬다.

    ④닉네임 중복검사를 한다.
        ㉠ 아이디와 상동

    ⑤비밀번호 패턴검사를 한다.
        ㉠ 비밀번호 기입시 해당패턴과 부합하는지 자동체크 후 아니라면 기입란 옆에 빨간 글씨로 경고문이 뜬다.

        ㉡해당조건이라면 기입란 옆에 사용가능을 알린다.

    ⑥비밀번호 확인을 한다.
        ㉠ 비밀번호 확인란에 비밀번호를 다시 기입 시 상단과 같은 비밀번호가 아니라면 기입란 옆에 빨간 글씨로 경고문이 뜬다.

        ㉡ 상단과 같은 비밀번호라면 올바른 비밀번호라는 경고문이 뜬다.

    ⑦가입버튼을 누른다.
        ㉠ 가입버튼 클릭 시 가입가능

        ㉡위 조건을 불만족 시 경고창이 발생

        ㉢중복검사 후 변경 한 경우가 있다면 경고창이 뜨며 가입이 불가능하다.

    ⑧리셋버튼을 누를 시 페이지가 초기화 된다.


▣ 로 그 인

    ① 아이디를 기입란에 입력한다.
  
    ② 비밀번호를 기입란에 입력한다.
  
    ③ 로그인 버튼을 누른다.
        ㉠ 로그인 버튼 클릭 시 아이디와 비밀번호가 일치하지 않는다면 경고창이 발생한다.
  
    ④ 로그인에 성공한다면 페이지가 갱신되고 로그인 창 대신 기본개인정보창과 게임스타트 버튼이 활성화 된다.


▣ 자 유 게 시 판
 
    ① 자유게시판 버튼 클릭

    ② 자유게시판의 공지글과 게시글들이 계층구조로 펼쳐진다

    ③ 리스트 하단에 페이징이 펼쳐진다.
        ㉠ 페이징 클릭 시 해당 페이지로 이동

    ④ '글쓰기' 버튼이 확인 가능하다.
        ㉠ 글쓰기 버튼 클릭
 
        ㉡ 글쓰기 페이지로 이동
 
        ㉢ 제목을 기입한다.
 
        ㉣ 내용을 기입한다. 

        ㉤ 등록 버튼 클릭 시 글 작성 완료, 자유게시판 리스트로 돌아온다.

        ㉥ 취소버튼 클릭 시 리스트로 돌아온다.
 
        ㉦ 리셋버튼 클릭 시 페이지가 초기화 된다. 
   
    ⑤ 게시글의 제목 클릭 시 읽기 기능 
        ㉠ 해당 게시글의 내용 확인이 가능하다

        ㉡ 해당 게시글의 작성자일 경우 수정, 삭제 버튼이 표시된다.
            ⓐ 수정버튼 클릭 시 수정 페이지가 열린다.

            ⓑ 열린 수정폼에는 기존 내용이 미리 기입되어 있다.
  
            ⓒ 버튼기능은 글쓰기 기능과 동일

       ㉢ 삭제 버튼 클릭시 확인 창이 팝업되고 확인 시 삭제 후 결과 알림창이 팝업되며
           리스트로 되돌아온다.

    ⑥ 검색란이 확인 가능하다
        ㉠ 검색란의 카테고리로 검색이 구분 가능하다
      
        ㉡ 기입란에 입력 후 엔터 혹은 검색버튼 클릭

        ㉢ 검색 후 리스트를 보여준다.

▣ 게임(1페이지, 대기방)
    
     ① 방 목록이 표시된다.
         ㉠ 방목록 클릭 시 해당 방으로 들어가진다.

     ② 방 목록 하단에 페이징이 표시된다.

     ③ 페이징 하단에 새로고침 버튼이 표시된다.
         ㉠ 새로고침 버튼 클릭 시 빈방우선순으로 정렬이 되어 새로 리스트가 불러진다.
 
     ④ 페이징 하단부에 전체 채팅란이 확인 가능하다.
         ㉠ 디폴트로 전체채팅이 가능하며 기입란 왼쪽에 특정유저의 아이디를 입력 후 채팅시 귓속말채팅이 가능하다.  

     ⑤ 방 목록 오른쪽에 유저/친구 탭 목록이 확인가능하다.
         ㉠ 목록의 유저 클릭 시 친구신청, 게임 초대 버튼 생성
  
         ㉡ 친구신청 클릭
             ⓐ친구 신청 버튼 클릭 시 본 유저의 친구 탭 목록에 해당 인원의 아이디가 추가된다.

         ㉢ 게임초대 버튼 클릭
             ⓐ 게임 초대 버튼 클릭 시 방제를 적고 완료를 누르면 상대방에게 초대 메시지가 전달

             ⓑ 해당 유저가 초대 메시지의 확인버튼을 누르면 방이 개설되며 해당 방으로 입장된다.
    
     ⑥ 방만들기 버튼이 확인 가능하다
         ㉠ 방만들기 버튼 클릭 시 방제를 입력

         ㉡ 확인 버튼을 클릭 시 방이 개설되며 방으로 입장된다.


▣ 게임(2페이지, 방)

     ① 유저 목록이 확인가능하다.
         ㉠ 유저 이름, 전적이 확인 가능하다.

     ② Ready버튼이 확인가능하다.
         ㉠ 방장이 아닌 유저가 Ready를 누른다.
        
         ㉡ 클릭 시 해당 유저목록에 표시된다.
 
     ③ 시작버튼이 확인가능하다.
         ㉠ 상대방이 Ready상태가 아닐 경우 비활성화

         ㉡ 스타트버튼 클릭 시 게임 페이지로 이동

     ④ 추방버튼이 확인 가능하다.
         ㉠ 유저 클릭 후 추방버튼 클릭 시 확인 창이 팝업되고 확인 시 추방

     ⑤ 채팅란이 확인 가능하다.
         ㉠ 방에 있는 유저화 채팅이 가능하다.


▣ 게임(3페이지, 인게임)

     ① 게임 시작 시 상대방과 본인의 게임화면이 보인다.

     ② 아군기와 타유저의 아군기 등장

     ③ 적출현, 탄환이 패턴에 맞게 접근해 온다.

     ④ 탄환, 적과 충돌 시 피해판정이 발생하고 HP가 소모된다.

     ⑤ 키보드의 방향키를 누르면 해당 방향으로 움직인다.

     ⑥ A, S버튼을 누르면 일반공격과, 특수공격이 가능하다.

     ⑦ 내려오는 버프아이템 획득 시 즉시 공격탄환의 공격력이 강해진다.

     ⑧ 방해아이템을 획득 시 즉시 타유저에게 방해효과가 가해진다.

     ⑨ 타유저 혹은 본인의 아군기의 HP가 모두 소모됬을 시 게임의 종료 알림이 발생하며

          게임이 종료된다.

     ⑩ 게임종료 시 게임 2페이지(방)으로 이동한다.
